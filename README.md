####项目说明 : 
终端俄语翻译工具
> [项目地址](https://coding.net/u/yihangwang/p/RussianTranslator/git)

---
####安装方法 :
1. 克隆项目
```
git clone https://git.coding.net/yihangwang/RussianTranslator.git
cd PyTranslator
```
2. 进行安装 
```
sudo python Setup.py
按照Setup.py中的指引就可以完成安装
```

---
####使用方法 : 
```
Usage : 
  fy [Your words]
Example : 
  fy help
  fy 帮助
  fy "Help me"
```
